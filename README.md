## Features

### Dashboard
- **General Attendance**: Overview of attendance across all disciplines.
- **Schedule**: Today's classes with accurate time slots.
- **Quick Access**: Shortcuts to lessons, announcements, and grades.

### Calendar
- **Views**: Toggle between Week and Month views.
- **Accuracy**: Precise time-grid visualization for classes (e.g., 8:00 - 9:30).
- **Details**: Shows room, duration, and type of event.

### Academic Management
- **Disciplines**: Detailed view of each subject including summary, professors, and schedule.
- **Lessons**: Dedicated page for each lesson with:
  - Video recording placeholder
  - Transcripts
  - Content/Materials (downloadable)
  - Tasks associated with the lesson
  - Participant list
- **Tasks**: Tracking of homework and projects with:
  - Status (Pending, Submitted, Late)
  - Full dates (Assigned vs Due)
  - Modality (Online vs Presencial)
- **Evaluations**: Detailed breakdown of grades with:
  - Weight percentages
  - Modality (Online vs Presencial)
  - Topics covered and allowed materials

### Communication
- **Email**: Full-featured internal email client with folders (Inbox, Sent, Drafts, Trash).
- **Chat**: Real-time messaging with support for:
  - Direct messages
  - Group chats
  - Call history (Voice/Video)
- **Contacts**: Directory of professors, colleagues, and support services with direct actions.

### Resources
- **Storage**: File management system with:
  - Categories (Folders, Files, Shared, Trash)
  - Type filters (Word, Excel, PowerPoint)
  - Sharing capabilities
- **Documents**: Creation and management of academic documents.
